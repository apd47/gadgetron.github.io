Detailed Description

BLAS level-1 functions on the cuNDArray class.

cuNDArray_blas.h provides BLAS level-1 functions on the cuNDArray class. The cuNDArray is temporarily reshaped to a column vector for the respective operations. The implementation is based on CUBLAS. This code is purposely split into a header and underlying implementation (.cpp) as this allows specific instantiation of the supported template types. The supported types are float, double, std::complex<float>, std::complex<double>, Gadgetron::complext<float>, and Gadgetron::complext<double>.
