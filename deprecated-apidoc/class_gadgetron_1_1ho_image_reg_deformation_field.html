Detailed Description

template<typename ValueType, unsigned int D>
class Gadgetron::hoImageRegDeformationField< ValueType, D >

deformation field is defined as hoNDImage the deformation field can be accessed on image pixels if the non-integer image pixels are used to access deformaiton field, an image interpolator is used linear interpolator is used for deformation field the unit of stored deformation field is in pixel, not in world coordinates
