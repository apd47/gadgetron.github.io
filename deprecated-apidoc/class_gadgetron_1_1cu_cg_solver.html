Detailed Description

template<class T>
class Gadgetron::cuCgSolver< T >

Instantiation of the conjugate gradient solver on the cpu.

The class cuCgSolver is a convienience wrapper for the device independent cgSolver class. cuCgSolver instantiates the cgSolver for type cuNDArray<T>.
