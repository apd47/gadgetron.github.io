Detailed Description

template<class T>
class Gadgetron::cuNDFFT< T >

Wrapper of the CUFFT library for ndarrays of type complext.

Wrapper of the CUFFT library for ndarrays of type complext<REAL>. The class' template type is a REAL, ie. float or double. The FFTs are performed in-place.
