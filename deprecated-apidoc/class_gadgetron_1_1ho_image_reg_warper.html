Detailed Description

template<typename ValueType, typename CoordType, unsigned int DIn, unsigned int DOut>
class Gadgetron::hoImageRegWarper< ValueType, CoordType, DIn, DOut >

warp the source image to the grid of target image under a transformation both image domain warpping and world coordinate warpping is implemented for the image domain warpping, the pixels are in the coordinate of image grid input and output can have different dimensions input has DIn dimension and output has DOut dimension
