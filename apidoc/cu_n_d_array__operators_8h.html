Detailed Description

Common element-wise arithmetic operators on the cuNDArray class.

cuNDArray_operators.h defines element-wise arithmetic array operations on the cuNDArray class. We define the common operators +=, -=, *= and \= for both array-array and array-constant operations. We have deliberately omitted to define operator+, operator- etc. since this would require returning an cuNDArray, in turn invoking an explicit memcpy by the assignment operator. Batch mode functionality is provided. The implementation is based on Thrust. This code is purposely split into a header and underlying implementation (.cu) as this allows specific instantiation of the supported template types. Furthermore thrust code can only be compiled by nvcc. The supported types are float, double, Gadgetron::complext<float> and Gadgetron::complext<double>. Scalars can be applied to complex numbers of corresponding precision.
