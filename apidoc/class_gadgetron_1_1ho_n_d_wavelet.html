Calculates the forward/inverse wavelet transform in: input array [RO E1 E2 ...] out: output array of wavelet coefficients level: number of wavelet transformation levels NDim: the number of transformation dimensions if NDim==1, 1D transformation is performed on the first dimension, out will have the size [RO 1+level E1 E2 ...] if NDim==2, 2D transformation is performed on the first two dimensions, out will have the size [RO E1 1+3*level E2 ...] if NDim==3, 3D transformation is performed on the first three dimensions, out will have the size [RO E1 E2 1+7*level ...] if forward==false, the role of in and out is switched and inverse wavelet transform is performed
